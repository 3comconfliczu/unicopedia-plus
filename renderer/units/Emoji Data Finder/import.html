Instructions
  • The Find by Name feature of the Emoji Data Finder utility displays a list of basic data (emoji, short name, keywords, code points) of matching Unicode emoji searched by name or keyword, including through regular expressions.
  • After entering a query, click on the Search button to display a list of all relevant matches, if any.
  • This feature deals with the 4,733 emoji defined in the Emoji 15.0 version of the emoji-test.txt data file.
  • The 9 component (5 skin-tone and 4 hair-style) emoji and the 3,655 fully-qualified (RGI) emoji are presented in a standard way, while the 1,069 non-fully-qualified emoji are shown in a distinctive muted (grayed out) style.
  • Use the Results ▾ pop-up menu to perform an action among:
    • Copy Results [copy the results as string to the clipboard]
    • Save Results.. [save the results as string to a text file]
    • Clear Results [clear the current list of results]
  • Various examples of regular expressions are provided for quick copy-and-paste.
  • Note: RGI stands for Recommended for General Interchange.
Regex Examples
  • Emoji whose keywords contain one character with the 'Ideographic' binary property:
    \p{Ideographic}
  • Emoji whose keywords contain one character with the 'Script' property value equal to 'Katakana':
    \p{Script=Katakana}
  • Emoji whose keyword is "pet":
    ^pet$
  • Emoji whose keywords contain either "!" or "?":
    [!?]
  • Emoji whose keywords contain four digits in a row:
    \d{4}
  • Emoji whose keywords contain three identical characters in a row:
    (.)\1\1
  • Emoji whose keyword is either "+1" (thumbs up) or "-1" (thumbs down):
    ^[+-]1$
  • Emoji whose keyword is a single mathematical symbol ("+", "−", "×", "÷"):
    ^\p{Math_Symbol}$
  • Emoji whose keyword contain one character belonging to the 'Other_Symbol' category; a check mark ("✓"), for instance:
    \p{Other_Symbol}
  • Emoji whose name contains both "flag" and "island" (but not "islands"):
    (?=.*flag)(?=.*island\b)
  • Emoji whose name or keyword is "koala" or "panda" or "rabbit" or "unicorn":
    ^(koala|panda|rabbit|unicorn)$

Regex Examples
  • Emoji containing a handshake:
    \u{1F91D} or 🤝
  • Emoji containing a kiss mark:
    \u{1F48B} or 💋
  • Emoji containing a red heart:
    \u2764\uFE0F or ❤️
  • Emoji starting with a white flag:
    ^\u{1F3F3}\uFE0F or ^🏳️
  • Emoji starting with a black flag:
    ^\u{1F3F4} or ^🏴
  • Emoji ending with a left speech bubble:
    \u{1F5E8}\uFE0F$ or 🗨️$
  • Emoji of keycap digits 0 to 9:
    ^[0-9]\uFE0F\u20E3$ or ^[0-9]\uFE0F⃣$
  • Emoji which are components, i.e., skin tones and hairstyles:
    ^\p{Emoji_Component}$
  • Emoji of country flags:
    ^\p{Regional_Indicator}\p{Regional_Indicator}$
    or
    ^[\u{1F1E6}-\u{1F1FF}][\u{1F1E6}-\u{1F1FF}]$
    or
    ^[🇦-🇿][🇦-🇿]$
  • Emoji of country flags made of two identical regional indicators:
    ^(\p{Regional_Indicator})\1$
    or
    ^([\u{1F1E6}-\u{1F1FF}])\1$
    or
    ^([🇦-🇿])\1$
  • Emoji of country flags, excluding macroregion flags:
    ^(?!\u{1F1EA}\u{1F1FA}|\u{1F1FA}\u{1F1F3})[\u{1F1E6}-\u{1F1FF}][\u{1F1E6}-\u{1F1FF}]$
    or
    ^(?!🇪🇺|🇺🇳)[🇦-🇿][🇦-🇿]$
  • Emoji of subdivision flags:
    ^\u{1F3F4}[\u{E0030}-\u{E0039}\u{E0061}-\u{E007A}]+\u{E007F}$
    or
    ^🏴[\u{E0030}-\u{E0039}\u{E0061}-\u{E007A}]+\u{E007F}$
  • Emoji starting with 'family: man, woman, girl':
    ^\u{1F468}\u200D\u{1F469}\u200D\u{1F467}
    or
    ^👨\u200D👩\u200D👧
    or
    ^👨‍👩‍👧
  • Emoji with a light skin tone:
    ^\p{Emoji_Modifier_Base}\u{1F3FB}
    or
    ^\p{Emoji_Modifier_Base}🏻
  • Emoji with a skin tone variant:
    ^\p{Emoji_Modifier_Base}\p{Emoji_Modifier}
    or
    ^\p{Emoji_Modifier_Base}[🏻-🏿]
  • Emoji of thumbs (up and down) with a skin tone variant:
    ^[👍👎]\p{Emoji_Modifier}$
    or
    ^[👍👎][🏻-🏿]$
    or
    ^[👍👎].$
  • Emoji with the 'Emoji' property, but without the 'Emoji_Presentation' property (non-fully-qualified):
    ^(?!\p{Emoji_Presentation})\p{Emoji}$
  • Emoji with the 'Emoji' property, but without the 'Emoji_Presentation' property (fully-qualified):
    ^(?!\p{Emoji_Presentation})\p{Emoji}\uFE0F$
  • Emoji with the 'Emoji_Modifier_Base' property, but without the 'Emoji_Presentation' property (non-fully-qualified):
    ^(?!\p{Emoji_Presentation})\p{Emoji_Modifier_Base}$
  • Emoji with the 'Emoji_Modifier_Base' property, but without the 'Emoji_Presentation' property (fully-qualified):
    ^(?!\p{Emoji_Presentation})\p{Emoji_Modifier_Base}\uFE0F$
  • Emoji made of a single code point belonging to the BMP (Basic Multilingual Plane):
    ^[\u0000-\uFFFF]$
  • Emoji made of a single code point belonging to a non-BMP "astral plane":
    ^[^\u0000-\uFFFF]$
  • Emoji made of exactly six or eight code points:
    ^......(..)?$ or ^(.{6}|.{8})$
Instructions
  • The Filter Text feature of the Emoji Data Finder utility displays in real time a list of basic data (emoji, short name, keywords, code points) of all the Unicode emoji contained in a text string.
  • Text can by directly typed, or pasted from the clipboard into the main input field.
  • It is also possible to input predefined sets of emoji selected from the Samples ▾ pop-up menu.
  • As a convenience, the input field can be emptied using the Clear button.
  • Use the Filter ▾ pop-up menu to perform an action among:
    • Discard Non-Emoji [strip out non-emoji characters]
    • Upgrade to RGI Emoji [restore incomplete emoji to their RGI form]
    • Remove Duplicate Emoji [delete emoji duplicates]
  • This feature deals with the 4,733 emoji defined in the Emoji 15.0 version of the emoji-test.txt data file.
  • The 9 component (5 skin-tone and 4 hair-style) emoji and the 3,655 fully-qualified (RGI) emoji are presented in a standard way, while the 1,069 non-fully-qualified emoji are shown in a distinctive muted (grayed out) style.
  • Use the Results ▾ pop-up menu to perform an action among:
    • Copy Results [copy the results as string to the clipboard]
    • Save Results.. [save the results as string to a text file]
